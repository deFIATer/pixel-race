# Pixel Race

Prosta gra wyścigowa multiplayer w stylu pixel art.

## Jak uruchomić

1. Zainstaluj zależności (jeśli jeszcze tego nie zrobiłeś):
   ```bash
   npm install
   ```

2. Uruchom serwer:
   ```bash
   npm start
   ```

3. Otwórz przeglądarkę i wejdź na adres:
   http://localhost:3000

Aby przetestować tryb multiplayer, otwórz grę w kilku kartach przeglądarki.

## Sterowanie

- **Strzałka w górę**: Przyspieszanie
- **Strzałka w dół**: Hamowanie / Cofanie
- **Strzałka w lewo/prawo**: Skręcanie
